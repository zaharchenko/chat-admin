# Редактор команд чата на Vue 3 + VueFlow

Проект для визуального редактирования команд чата-бота с использованием Vue 3 и VueFlow.

## Функционал

- **Список команд**: Просмотр всех команд чата в боковой панели
- **Редактирование команд**:
  - ID команды
  - Название
  - Ключевые слова
  - Статус включения (enabled)
  - Тип команды (основная/дочерняя)
  - Текст ответа
  - Дочерние команды (связи)
- **Визуальная схема**: Отображение связей между командами в виде графа (VueFlow)
- **Импорт/Экспорт JSON**: Сохранение и загрузка конфигурации команд

## Установка

```bash
cd chat-command-editor
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Структура данных

Каждая команда имеет следующую структуру:

```json
{
  "id": "unique_id",
  "name": "Название команды",
  "keywords": ["ключевое", "слово"],
  "enabled": true,
  "main": true,
  "response": "Текст ответа",
  "childs": ["id_дочерней_команды"]
}
```

## Использование

1. Выберите команду из списка слева или кликните на узел в диаграмме
2. Отредактируйте поля в форме редактора
3. Добавьте связи с другими командами через секцию "Дочерние команды"
4. Сохраните изменения через кнопку "Сохранить JSON"

## Технологии

- Vue 3 (Composition API)
- VueFlow (@vue-flow/core)
- Vite
